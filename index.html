Ctxfmt

A program for formatting LaTeX as ConTeXt, powered by DeepSeek and compatible with Simext.

Download: github.com/EIPIofficial/Ctxfmt

Usage

  1. Submit a PDF to DeepSeek with DeepThink (R1) to convert it into LaTeX.
  2. Create a .tex document and paste the output.
  3. Run ctxfmt.lua using Lua: ctxfmt.lua input.tex output.tex
  4. Add \environment[...] to output.tex and compile with ConTeXt.
  5. Note: The fig.lua module is not yet complete — add figures manually for now.
